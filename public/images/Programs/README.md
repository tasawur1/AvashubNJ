# Programs Page Images

Use this folder for Programs page desktop/tablet images.

## Hero Images

- `hero-banner-programs.png` - desktop/tablet Programs hero banner
- `hero-banner-programs-mobile.png` - mobile Programs hero banner

## Mobile Images

Mobile-only section images should go in:

`public/images/Programs/mobile/`

See `public/images/Programs/mobile/README.md`.

## Note

Replacing an existing file with the same filename needs no code change. New filenames must be added to the image config.
