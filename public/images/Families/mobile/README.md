# Families Mobile Section Images

Use this folder for mobile-only Families page section images.

Recommended filenames:

- `hero-banner-mobile.png` - top mobile hero image
- `family-centered-care.png` - Family-Centered Care support card image, if used
- `education-empowerment.png` - Education & Empowerment support card image, if used
- `emotional-support.png` - Emotional Support support card image, if used
- `community-connection.png` - Community Connection support card image, if used
- `ongoing-communication.png` - Ongoing Communication support card image, if used
- `family-journey.png` - Every Step of the Journey section image, if used
- `parent-caregiver-training.png` - Parent & Caregiver Training card image, if used
- `you-deserve-support-too.png` - You Deserve Support Too card image, if used
- `helpful-resources.png` - Helpful Resources card image, if used

## How To Update

Replace a file using the same filename when you want no code changes.

If you add a new filename, update the image config so the page points to it.
