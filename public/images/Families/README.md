# Families Page Images

Use this folder for Families page desktop/tablet images.

## Hero Images

- `hero-banner-families.png` - desktop/tablet Families hero banner
- `hero-banner-families-mobile.png` - mobile Families hero banner

## Mobile Images

Mobile-only section images should go in:

`public/images/Families/mobile/`

See `public/images/Families/mobile/README.md`.

## Note

Replacing an existing file with the same filename needs no code change. New filenames must be added to the image config.
