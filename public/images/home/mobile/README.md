# Home Mobile Section Images

Use this folder for mobile-only Home page section images.

Recommended filenames:

- `hero-banner-mobile.png` - top mobile hero image
- `real-life-skills.png` - Real-Life Skills value card image, if used
- `confidence.png` - Confidence value card image, if used
- `independence.png` - Independence value card image, if used
- `belonging.png` - Belonging value card image, if used
- `relationship-based-care.png` - Relationship-Based Care value card image, if used
- `kids.png` - Programs for Every Stage Kids card
- `school-age.png` - Programs for Every Stage School-Age card
- `teens.png` - Programs for Every Stage Teens card
- `young-adults.png` - Programs for Every Stage Young Adults card
- `bottom-banner.png` - bottom emotional/CTA image, if used

## How To Update

Replace a file using the same filename when you want no code changes.

If you add a new filename, update the image config so the page points to it.
