# Home Page Images

Use this folder for Home page desktop/tablet images.

## Current Images

- `hero-banner.png` - desktop/tablet home hero banner
- `hero-banner-mobile.png` - mobile home hero image currently used on the mobile homepage
- `kids-program.png` - Kids program card image
- `school-age-program.png` - School-Age program card image
- `teens-program.png` - Teens program card image
- `young-adults-program.png` - Young Adults program card image
- `after-school-1.png` - after-school/support image 1
- `after-school-2.png` - after-school/support image 2
- `after-school-3.png` - after-school/support image 3
- `after-school-4.png` - after-school/support image 4
- `bottom-banner.png` - bottom banner / final CTA image

## Mobile Images

Mobile-only image filenames should go in:

`public/images/home/mobile/`

See `public/images/home/mobile/README.md`.

## Note

Replacing an existing file with the same filename needs no code change. New filenames must be added to the image config.
