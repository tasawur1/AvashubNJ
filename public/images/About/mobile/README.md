# About Mobile Section Images

Use this folder for mobile-only About page section images.

Recommended filenames:

- `hero-banner-mobile.png` - top mobile hero image
- `built-from-real-life.png` - "Built From Real Life" section
- `founder-story.png` - "Meet the Heart Behind Ava's Hub" / Founder Story section
- `children.png` - "Who We Serve" Children card
- `teens.png` - "Who We Serve" Teens card
- `young-adults.png` - "Who We Serve" Young Adults card
- `families.png` - "Who We Serve" Families card
- `community-belonging.png` - "You Were Never Meant To Do This Alone" section

## How To Update

Replace an image using the same filename when you want no code changes.

If you add a new filename, update `src/data/images.ts` so the About page points to it.
