# About Page Images

Use this folder for About page desktop/tablet images.

## Hero Images

Replace these files when updating the About hero banner:

- `hero-banner-about.png` - desktop/tablet About hero banner
- `hero-banner-about-mobile.png` - mobile About hero banner

## Mobile Section Images

Mobile-only section images should go in:

`public/images/About/mobile/`

See `public/images/About/mobile/README.md` for the mobile section filenames.

## Note

After adding new image files, the site may need a code update if the filename changes. If you replace an existing file using the same filename, no code change is needed.
