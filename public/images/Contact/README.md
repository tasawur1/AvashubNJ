# Contact Page Images

Use this folder for Contact page desktop/tablet images.

## Hero Images

- `hero-banner-contact.png` - desktop/tablet Contact hero banner
- `hero-banner-contact-mobile.png` - mobile Contact hero banner

## Mobile Images

Mobile-only section images should go in:

`public/images/Contact/mobile/`

See `public/images/Contact/mobile/README.md`.

## Note

Replacing an existing file with the same filename needs no code change. New filenames must be added to the image config.
