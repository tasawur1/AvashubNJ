# Contact Mobile Section Images

Use this folder for mobile-only Contact page section images.

Recommended filenames:

- `hero-banner-mobile.png` - top mobile hero image
- `get-in-touch.png` - Get in Touch card image, if used
- `send-message.png` - Send Us a Message form image, if used
- `find-us-map.png` - Find Us map image or placeholder
- `ready-next-step.png` - Ready to Take the Next Step CTA image, if used
- `support-feature-strip.png` - support feature strip image, if used
- `closing-cta.png` - large purple closing CTA image, if used

## How To Update

Replace a file using the same filename when you want no code changes.

If you add a new filename, update the image config so the page points to it.
