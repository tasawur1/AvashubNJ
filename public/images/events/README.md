# Events Page Images

Use this folder for Events page desktop/tablet images.

## Hero Images

- `hero-banner-events.png` - desktop/tablet Events hero banner
- `hero-banner-events-mobile.png` - mobile Events hero banner

## Current Highlight Images

- `column-1.png` - Upcoming Highlights card 1
- `column-2.png` - Upcoming Highlights card 2
- `column-3.png` - Upcoming Highlights card 3
- `column-4.png` - Upcoming Highlights card 4

## Mobile Images

Mobile-only section images should go in:

`public/images/events/mobile/`

See `public/images/events/mobile/README.md`.

## Note

Replacing an existing file with the same filename needs no code change. New filenames must be added to the image config.
