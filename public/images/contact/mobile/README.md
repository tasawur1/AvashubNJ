# Contact Mobile Section Images

Use this folder for mobile-only Contact page section images.

Current filenames:

- `hero-banner-contact-mobile.png` - top mobile hero image
- `contact-support-image.png` - optional support/contact section image
- `contact-map-image.png` - optional Find Us/map section image
- `newsletter-contact.png` - Stay Connected newsletter image
- `final-cta-contact.png` - optional final CTA image

## How To Update

Replace a file using the same filename when you want no code changes.

If you add a new filename, update the image config so the page points to it.
